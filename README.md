# Unity_Game
This will become a minimalistic game designed with Unity and C#
The gameplay is a rougue like game inspired by vampire survivors.

Functionalities implemented:
- Player movement
- Enemy Spawning
- Weapon Rotation
- Enemy Killing and Respawning with increased movement speed
- Potion spawning (without any effect yet)

To be implemented:
- Score system
- Potion effects
- Weapon Pick Up
- new Enemy Type
